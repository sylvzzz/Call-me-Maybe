# Call-me-Maybe
Introduction to Function calling and parsing with LLM's

# Instructions

## Installation

```bash
# in 42 we dont have a lot of space in home folder so replace user_42 with your intra user

curl -LsSf https://astral.sh/uv/install.sh | UV_INSTALL_DIR=/sgoinfre/$(USER)/.local/bin sh

export UV_CACHE_DIR=/sgoinfre/$(USER)/.cache/uv
export PIP_CACHE_DIR=/sgoinfre/$(USER)/.cache/pip

```
